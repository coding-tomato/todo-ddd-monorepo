# Text List Manager Application

## Project Description

It is required to build the layout (HTML and CSS) and develop (JavaScript) an application to manage a list of text strings.

**Design Reference:** https://xd.adobe.com/view/ea696dd0-8781-4460-8720-36deb2d19b2a-bf3a/

---

## Functional Specification

### Required Components

The application must have a user interface that includes, at least, the following elements:

- A container where text strings will be added or deleted
- A text input field where the user can write texts they wish to add to the list
- A button to add new entries
- A button to delete items from the list

### Functionalities

The application must implement the following functionalities:

1. **Add text entries**
   - Allow the user to write and add text entries to a list
   - Empty entries cannot be added

2. **Delete items from the list**
   - List items must be selectable by the user
   - Items cannot be deleted without having selected one or more elements
   - The ability to select multiple items at once is not required (but will be appreciated if implemented)
   - **Desirable (optional):** Allow deleting items by double-clicking on the item

3. **Undo changes**
   - **Desirable (optional):** Allow the user to undo, at minimum, the last change made via an undo button

---

## Technical Specification

### Development Requirements

It is desirable to create a solution using Vanilla JS and another using React. If difficulties arise with Vanilla JS development, only the React solution can be delivered.
